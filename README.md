# 🏠 StudyNook Server Side (A9)

Backend API for the StudyNook room booking platform.  
Built with **Node.js, Express, MongoDB, JWT authentication**, and deployed on **Vercel (serverless)**.

---

## 🚀 Live Server
👉 [https://studynook-server-alpha.vercel.app](https://studynook-server-alpha.vercel.app)

---

## 📌 Features

- 🔐 JWT Authentication (cookie-based)
- 🏠 Room management API (CRUD via routes)
- 📅 Booking system with time conflict prevention
- 👤 User-specific bookings
- 🔒 Protected routes with middleware
- 🌐 CORS configured for frontend integration
- ☁️ MongoDB Atlas integration
- ⚡ Serverless deployment ready (Vercel)

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB (Native Driver)
- JSON Web Token (JWT)
- Cookie Parser
- CORS
- dotenv
